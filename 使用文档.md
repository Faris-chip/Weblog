****以后更新博客**
以后你每次写完文章，只要：**

`git add -A `

`git commit -m "Update blog" `

`git push`

**怎么预览和生成**
本地预览：

`npm run server`

生成静态文件：

`npm run build`

如果发现旧内容没刷新，可以先：

`npm run clean`

`npm run build`

**标签怎么增加或删减**
标签不是单独维护的，它是从文章里自动统计出来的。

增加标签：

* 在文章 front-matter 里给 **tags** 增加一项即可

删减标签：

* 去相关文章的 **tags** 里删掉那个标签
* 如果所有文章里都不再使用这个标签，重新生成后它就会从标签页消失

然后重新 **npm run build**，这个标签的关联就会减少；如果没有任何文章再使用它，它对应页面也会消失。

**一个实用建议**
后面你每写一篇文章，尽量固定这几项：

* **categories** 只放 1 个主分类
* **tags** 放 2 到 5 个关键词
* 配一张 **index_img**
